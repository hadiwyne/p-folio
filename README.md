# Portfolio Website

A personal portfolio static website.

## Features

- **Dark Theme**: Dark colour scheme
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Navigation with smooth scroll behavior

```
├── scripts/

│   └── data.js         # Projects data structure
├── assets/
│   └── resume.pdf      # My resume
└── README.md          
```
